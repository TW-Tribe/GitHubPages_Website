## Video and Images
Adding images or video.

Adding video and hyperlink follow the same process while adding image is separated with the usage of an exclamation mark. 

in a Markdown file follows almost the same rule that you follow to add a hyperlink. You do not insert images, but provide the path to the image folder.


### Add a Video
Fetch the correct URL and place with the shortcode. Fill the details and get the text hyperlinked to open in a new tab.
